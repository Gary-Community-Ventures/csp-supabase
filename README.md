# Local Supabase Development

## Start local Supabase

```bash
make start
```

## Get credentials

```bash
make env
```

## Access local Supabase Studio

```bash
make studio
```

## Pull remote schema

Can be used if there are changes made to the prod db that we need locally.

```bash
make pull
```

## Stop local Supabase

```bash
make stop
```
